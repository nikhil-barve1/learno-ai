# LearnoAI - Project Specification

## Hackathon Objective

Build a polished AI-powered learning consultant that generates personalized learning roadmaps.

Submission deadline is the highest priority.

Ship first.

Improve later.

---

# Problem Statement

Many learners know what they want to become but don't know:

- where to start
- what to learn next
- which free resources to use
- how much time it will take
- what projects to build
- how to prepare for interviews

LearnoAI solves this.

---

# Target Users

- Students
- Freshers
- Career Switchers
- Self-learners

---

# MVP Features

## Landing Page

- Hero Section
- Product Description
- CTA

---

## AI Form

Collect:

- Name
- Current Qualification
- Current Experience
- Technology to Learn
- Current Skill Level
- Available Months
- Hours per Day

---

## AI Report

Generate:

- Personalized Learning Roadmap
- Weekly Study Plan
- Learning Strategy
- Official Resources
- Free Learning Resources
- Interview Preparation
- Practice Projects
- Beginner Mistakes
- Final Capstone Project

---

## UI

Dark Theme

Responsive

Professional

Fast

---

# Out of Scope

Do NOT build:

- Login
- Database
- Payment
- Dashboard
- History
- Analytics
- Notifications
- Progress Tracking

---

# Success Criteria

A user should be able to:

1. Open LearnoAI.
2. Fill the form.
3. Click Generate.
4. Receive a beautiful personalized roadmap.

Nothing else is required for the MVP.
